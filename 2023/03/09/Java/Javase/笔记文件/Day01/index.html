Java注释

1.注释

写源代码的时候，需要一些文字说明，使用注释来辅助说明（在使用的时候，编译器会忽略它的存在）

2.关键字

Java关键字是对Java编译器有特殊含义的字符串，是编译器和程序员的一个约定，程序员利用关键字来告诉编译器其声明的变量类型、类、方法特性等信息。Java语言共定义了如下所示的关键字。

3.标识符

其实就是名称，定义类，包，方法，变量名。为了一个好的命名

标识符的组成

  • 英文大小写字母 （A-Z a-z）

  • 数字 0-9

  • 符号_与￥

标识符的规则

  • 不能使用数字为开头

  • 不能使用关键字

  • 严格区分大小写，要做到见名知意

  • 驼峰命名法
